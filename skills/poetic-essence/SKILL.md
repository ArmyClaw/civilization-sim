# Poetic Essence - 诗意本质

## 概述
使用中国古典诗词来凝练项目的本质。古体诗以极简承载极大——20字可纳2000言。

## 格式规范

| 规范 | 要求 |
|------|------|
| 风格 | 古体诗（五言/七言为主） |
| 押韵 | 尽量押韵，意先于韵 |
| 行数 | 4句（绝句）或8句（律诗） |
| 诗源 | 仅中国古典诗词 |

## 三种模式

### 1. WRITE（诗意定调）
优先引用契合的古诗；若无完全契合者，化用古诗意象/句式适配场景。

化用原则：借古诗骨架（句式、韵律、意象结构），替换关键词，保留神韵但意思全新。

化用示例：
| 原诗 | 化用 | 场景 |
|------|------|------|
| 沉舟侧畔千帆过 | 改道侧畔千帆过 | 项目多次转型 |
| 行到水穷处，坐看云起时 | 行到用户穷，坐看需求起 | 用户调研瓶颈 |
| 不识庐山真面目 | 不识产品真面目 | 看不清产品本质 |
| 千磨万击还坚劲 | 千次迭代还坚劲 | 需求频繁变更 |

### 2. READ（借古喻今）
直接引用中国古诗，原封不动，解读当下处境。

### 3. INTERPRET（品诗得道）
深度品读诗词，提取行动启示。

## 算法

1. 吸收 — 理解项目处境和张力
2. 选择模式 — WRITE/READ/INTERPRET
3. 凝练 — 找契合古诗或化用
4. 关联 — 这首诗如何照亮当前处境？有什么行动启示？

## 诗词库参考

### 困境类
- 山重水复疑无路，柳暗花明又一村（陆游）
- 行到水穷处，坐看云起时（王维）
- 欲渡黄河冰塞川，将登太行雪满山（李白）

### 坚持类
- 千磨万击还坚劲，任尔东西南北风（郑燮）
- 路漫漫其修远兮，吾将上下而求索（屈原）
- 长风破浪会有时，直挂云帆济沧海（李白）

### 突破类
- 不识庐山真面目，只缘身在此山中（苏轼）
- 欲穷千里目，更上一层楼（王之涣）

## 常见错误
| 错误 | 修正 |
|------|------|
| WRITE模式原创打油诗 | 化用古诗骨架，借骨换肉 |
| 诗太长 | 绝句4句足矣 |
| 解读太牵强 | 只用自然契合的诗 |
| 只有文艺没有行动 | 必须有"对项目的启示" |
| 用外国诗 | 仅用中国古典诗词 |
