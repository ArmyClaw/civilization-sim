# 🏛️ Civilization Sim — 数字文明模拟器

> 模拟有经济、人口、资源、信仰、科技的虚拟文明，从采集野果到星际旅行。

## 阶段规划

### Phase 1: 基础生态
- [ ] 地图生成（程序化地形）
- [ ] 资源分布（食物、水、矿物、木材）
- [ ] 人口基本模型（出生/死亡/迁移）
- [ ] 简单采集行为

### Phase 2: 经济系统
- [ ] 物品交换与价值
- [ ] 专业分工（猎人/农民/工匠）
- [ ] 货币体系演化
- [ ] 供需平衡

### Phase 3: 社会结构
- [ ] 部落形成与首领
- [ ] 信仰体系
- [ ] 文化传播
- [ ] 冲突与战争

### Phase 4: 科技树

- [ ] 依赖关系（先有陶器才能有冶炼）
- [ ] 技术扩散
- [ ] 工业化与信息化

### Phase 5: 高级交互
- [ ] 外交系统

- [ ] 环境影响
- [ ] 文明评分与结局
