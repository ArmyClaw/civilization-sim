# 经济参数表规格 (Economy Parameters Specification)

## 概述

本规格文档定义文明模拟游戏中各种资源的基础价值、稀缺性系数以及相关经济参数。

## 物品类型定义

### 基础物品类型

| 物品类型 | 中文名称 | 基础价值 | 质量范围 | 耐久度范围 | 初始稀缺系数 |
|---------|---------|---------|---------|-----------|-------------|
| FOOD | 食物 | 1.0 | 0.1-2.0 | 0.5-1.0 | 1.0 |
| WATER | 水 | 1.2 | 0.1-2.0 | 0.8-1.0 | 1.0 |
| WOOD | 木材 | 0.8 | 0.5-2.0 | 0.3-1.0 | 1.2 |
| TOOL | 工具 | 3.0 | 0.8-2.0 | 0.2-1.0 | 0.8 |
| WEAPON | 武器 | 5.0 | 0.5-2.0 | 0.1-0.8 | 0.6 |
| SHELTER | 庇护所 | 8.0 | 0.8-2.0 | 0.9-1.0 | 0.5 |
| CLOTHING | 衣物 | 2.5 | 0.6-2.0 | 0.4-1.0 | 0.9 |
| MEDICINE | 药物 | 6.0 | 0.3-2.0 | 0.2-0.9 | 0.7 |

## 价值计算公式

### 基础价值计算
```
基础价值 = 物品类型基础价值 × 质量系数 × 耐久度系数
```

### 市场价值计算
```
市场价值 = 基础价值 × 供需平衡因子
```

## 供需平衡系统

### 供需平衡因子定义

| 供需状态 | 供应量 | 需求量 | 平衡因子 | 说明 |
|---------|-------|-------|---------|------|
| 严重供过于求 | > 150% | < 50% | 0.5 | 价值减半 |
| 轻微供过于求 | 100%-150% | 50%-100% | 0.7-0.9 | 价值小幅下降 |
| 供需平衡 | 80%-120% | 80%-120% | 0.9-1.1 | 价值基本稳定 |
| 轻微供不应求 | 50%-80% | 100%-150% | 1.1-1.3 | 价值小幅上升 |
| 严重供不应求 | < 50% | > 150% | 1.5-2.0 | 价值大幅上升 |

### 价格波动限制
- 单次最大价格波动：20%
- 价格波动平滑系数：0.7（保持70%旧价格 + 30%新价格）
- 最低价格限制：0.1（避免价值为负）

## 稀缺性系统

### 初始稀缺系数
基于物品在文明发展中的重要性，设置初始稀缺系数：

| 物品类型 | 稀缺系数 | 说明 |
|---------|---------|------|
| SHELTER | 0.5 | 生存必需，生产复杂，相对稀缺 |
| WEAPON | 0.6 | 安全保障，技术要求高 |
| MEDICINE | 0.7 | 健康保障，原材料稀缺 |
| TOOL | 0.8 | 生产工具，技术依赖性强 |
| CLOTHING | 0.9 | 生活品质改善，相对容易获得 |
| WATER | 1.0 | 基础必需，供应相对稳定 |
| FOOD | 1.0 | 基础必需，生产方式多样 |
| WOOD | 1.2 | 自然资源相对丰富 |

### 动态稀缺性调整
根据以下因素动态调整稀缺系数：
1. 技术发展水平：生产技术提升降低稀缺系数
2. 环境条件：干旱地区水资源稀缺系数增加
3. 人口密度：高密度地区食物稀缺系数增加
4. 季节变化：某些资源在不同季节稀缺性不同

### 生产成本公式
```
实际成本 = 基础成本 / (1 + 技术等级 × 0.2) × 质量系数
```

## 市场机制

### 价格历史记录
- 保存最近100次交易记录
- 记录格式：`(价格, 交易量)`
- 用于价格趋势分析和预测

### 交易规则
1. 价格浮动范围：市场价的80%-120%
2. 最小交易单位：1个物品
3. 交易成功率：基于价格合理性、双方资源状态
4. 交易税收：交易金额的5%（用于公共项目）
