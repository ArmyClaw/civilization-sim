## 首页新闻原文链接跳转修复 (多次 ~20:07-20:45)
- **问题1**：`card::after` 伪元素（200%×200% 旋转渐变装饰层）pointer-events 默认 auto，吞掉所有点击 → 改成 none
- **问题2**：加的 JS click handler 反而搞坏了 `<a>` 原生跳转，`window.open` 被手机浏览器当弹窗拦截 → 删掉 JS handler
- **问题3**：鸿蒙浏览器可能对伪元素 pointer-events 支持有差异 → card-source 加 z-index: 2
- 教训：不要给 `<a>` 标签加多余的 JS click handler，原生行为最可靠
