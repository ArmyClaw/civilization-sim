# 2026-06-20 Daily Notes

## 每日奇境网站修复（重要！）

### 问题：页面白屏/无样式
- **根因**：`<link rel="icon" href="data:image/svg+xml,<svg...>">` 内联SVG icon导致HTML解析器提前闭合，后续CSS link被吞掉，整个页面无样式
- **修复**：改为 `data:image/svg+xml;base64,...` 编码
- **教训**：永远不要在 `<link href>` 里直接内联裸SVG，用base64

### 问题：study子页面被每小时cron覆盖
- `db.py export` 本身只导出 lab-progress.json，不碰子页面
- 实际是cron AI在implement/deploy步骤时"热心"重写了study子页面
- 修复：cron prompt加了铁律禁止修改 study/*/index.html
- 已用git恢复到昨天好看的版本

### 问题：CSS class名不匹配
- 今天"奢华鎏金"CSS用了 `.content-card`/`.content-grid` 等新class
- app.js 生成的是 `.card` 系列旧class
- 两边对不上导致卡片无样式
- 已修复
